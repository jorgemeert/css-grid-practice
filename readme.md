Este é um projeto prático desenvolvido para consolidar conhecimentos em **CSS Grid Layout**. O objetivo foi criar uma interface de vitrine de produtos (e-commerce) que seja organizada, visualmente limpa e estruturada de forma semântica.

## 🚀 Tecnologias e Conceitos Aplicados

Neste projeto, foquei em aplicar conceitos modernos de desenvolvimento web:

* **HTML5 Semântico:** Estruturação correta de itens e seções.
* **CSS Grid Layout:** * Uso de `grid-template-columns` para criar a malha principal.
    * Posicionamento preciso de elementos (título, imagem, preço e descrição) dentro de cada card utilizando `grid-column` e `grid-row`.
    * Alinhamento de conteúdo com `place-self` e `justify-content`.
* **Design Responsivo:** Preparação para layouts que se adaptam a diferentes telas.

## 📸 Demonstração do Layout

O projeto consiste em uma grade de produtos onde cada item contém:
- Título centralizado
- Imagem do produto
- Descrição detalhada
- Preço em destaque
- Botão de ação (Call to Action) estilizado
